Python API for lirc
===================

This is a work in progress, don't even think about using it!


Installation
------------
::

    $ git clone https://github.com/leamas/python-lirc-api
    $ cd python-lirc-api
    $ python3 setup.py sdist
    $ cp dist/lirc*gz  somewhere/else.
    $ cd somewhere/else;
    $ pip install lirc*gz


Documentation
-------------

You need Doxygen to build the docs. Do:
::

    $ doxygen

to build api-docs.
